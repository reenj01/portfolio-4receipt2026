Ocan AI

This was my first hackathon and I'm so grateful to have won Meta's Best Use of AI in a Mixed Reality App on Meta Quest and received honorable mention for Game Prototyping Track at Reality Hack at MIT (Co-organized by Reality Hack, Inc. and VR/AR MIT)!!


Just in 2 days and 3 hours, with an all nighter on the last day, we built and designed Ocan, a two player co-op XR game where human intention is filtered through an AI intermediary, revealing how trust, agency, and consequence shift when machines mediate our shared dreams.


After months of wanting to work with the Arduino Uno Q, designed in collaboration with Qualcomm, I finally had the opportunity to build with it for this project. Using Meta Quest 3 and Arduino Uno Q, we built Ocan as a tightly integrated mixed reality game that relies on 3 core AI interactions: data transfer from Arduino to Unity using tcp web server, OpenAI integration in Unity using Meta LLM SDK to display AI generated text, and voice detection model created using Edge Impulse running on Arduino Uno Q.
